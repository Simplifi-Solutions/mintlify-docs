---
title: "How to Set Up User Permissions for a Department Manager"
---

The following is only an example of what is typically set up for department managers. Your store can provide more or less access as per your requirements

<Frame>
  ![Image 45 1024x813](/images/image-45-1024x813.png)
</Frame>

 1. Select **Our Profile**
 2. Select **Our Users**
 3. Select **Edit **next to the user you wish to edit
 4. Make sure this is **YES **– Active, if you ever need to revoke access then you can switch this toggle to **NO** and **Save**, making the user inactive
 5. Switch this toggle to **YES **to restrict the user to publishing casual jobs for their departments only
 6. Switch this toggle to **YES **to allow the user to access the rostering module
 7. Switch this toggle to **YES **to allow the user to approve or modify timesheets
 8. Switch this toggle to **YES **to allow the user to manage leave requests
 9. Select the **department** they will be able to access. According to your settings from steps 5-7, this user will be able to manage rosters, approve timesheets, and manage leave for all the departments that are selected here
10. Select all the departments this user will be able to view the **schedule **for – This is a view-only access, they will not be able to make modifications
11. Select **Save User Details**
