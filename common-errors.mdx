## Required Job Missing

Required job missing means that there is a job that needs to be scheduled in the [job queue](/leave-management-overview). To fix this error, go into the job queue and schedule the specified job.

## Name Matched Required

The NZ Teachers Council Registration contains a name alongside a registration number. If the user enters the wrong name or number then the registration details will not match up and therefore cause a name match error.
