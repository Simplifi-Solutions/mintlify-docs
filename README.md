# Simplifi Documentation

_All hosted in Mintlify, written in Markdown._

To differentiate between the public and private docs, ensure that any public pages are under the simplifi folder, and that they have public = true on their folder in the docs.json file.

Mintlify automatically has an agent run through the documentation when you hit publish, and if any broken links or spelling mistakes are detected, it will create a new branch with these errors fixed. Merge it if you'd like to.
