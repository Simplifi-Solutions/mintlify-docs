The Payroll Rules Engine is Simplifi's logic that your system uses to decide how, when, and how much to pay each person. Simplifi automates this logic by applying the right rule to every timesheet, shift, and contract and allows you to define custom rules and logic for specific events.

Start by filling in the name and validity dates.

**Who**

<Frame>
  ![2026 03 03 15 56 21](/images/2026-03-03_15-56-21.png)
</Frame>

Who determines which people the rule will apply to. Select which rule is appropriate using the drop-downs. Remember that multiple rules can be created.

**When**

<Frame>
  ![2026 03 03 15 58 46](/images/2026-03-03_15-58-46.png)
</Frame>

When determines the trigger for this rule to be applied. Multiple triggers can be made.

**Then**

<Frame>
  ![2026 03 03 16 00 00](/images/2026-03-03_16-00-00.png)
</Frame>

Then determines what action to take based on the previous rules.
