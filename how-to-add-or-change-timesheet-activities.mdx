---
title: "How to Add or Change Timesheet Activities"
---

The shift is added with the default activity set for the role, with a lunch break if entitled. You may wish to add or change these timesheet activities. To do so:

<Frame>
  ![Image 3 1024x349](/images/image-3-1024x349.png)
</Frame>

1. Select the shift bar
2. Select the clock icon to bring up the timesheet

<Frame>
  ![R2](/images/r2.png)
</Frame>

1. Click this + button to add an activity to the top of the list
2. Select one of these + buttons to add an activity directly below
3. Choose from the list of activities
4. Enter the finish time of this activity
5. If you wish to notify the staff member of the change, switch this toggle to a yes
6. Select Save Details.
