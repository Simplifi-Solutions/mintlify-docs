Under our users, you can manage, add and remove admin users as well as managers. To reset the password of a user, press reset password. To edit a user, press edit and adjust values accordingly.

<Frame>
  ![2026 03 09 13 33 39](/images/2026-03-09_13-33-39.png)
</Frame>

By pressing the managers tab, you can select managers for different departments as well as the organisation's manager.

You can also create new **Reporting Roles** in this page by pressing add a new reporting role. Reporting Roles are roles that managers/roles that aren't bound to a specific department.
