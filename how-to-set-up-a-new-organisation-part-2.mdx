## About

The Organisation is the business or organisation that employs and manages the users in Simplifi. This is the second part to setting up an organisation.

## Setting Up Guide

You can follow this guide to set up an organisation. This guide will run through heading by heading and detail what the settings are, what they do and provide an example of the setting. If a setting is complicated, it will lead to another page.

<Warning>
  Some options are omitted in the guide, these are the self-explanatory checkboxes and information fields
</Warning>

13. **Leave Requests**

| Setting Name                                                                   | Description                                                                                                                                                                                                                                                        |
| ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Has Leave Requests module enabled                                              | Ticking this box enables the leave requests module                                                                                                                                                                                                                 |
| Leave Endorsements enabled                                                     | If this is ticked then the leave requests are approved and managed at the head office instead of the individual branches                                                                                                                                           |
| Each Room/Department needs to approve requests that cover multiple departments | If the employee is split across multiple departments then each department manager will have to approve the leave                                                                                                                                                   |
| Show annual leave projected balances                                           | If this setting is turned on when approving leave managers will be able to view projected balances of annual leave                                                                                                                                                   |
| Show projected balances to staff                                                 | If this setting is turned on staff will also be able to see their projected balances                                                                                                                                                                               |
| Default leave entitlement                                                      | Default number of weeks leave entitlement for each employee                                                                                                                                                                                                        |
| Leave Balance Warnings enabled                                                 | If this is turned on, then staff will receive a warning pop-up if they try to apply for leave that exceeds their balance                                                                                                                                              |
| Public Holidays based on organisation address rather than the staff addresses  | Public holidays that are specific to different regions (e.g. Canterbury Anniversary Day) are applied based on the address specified in the business's address rather than the employee's address (e.g. business is in Canterbury but employee's address is in Otago) |

14. **External Timesheets**

Upon completing this guide, you will have entered all the settings for a new organisation in Simplifi. **Make sure to save your work!** Press **Save Organisation Details** at the bottom to save the settings. The organisation is now set up!
