title: "The Developer Environment"
---

You can download the developer environment from the Simplifi Launcher.

<Frame>
  ![2026 03 30 16 06 47](/images/2026-03-30_16-06-47.png)
</Frame>

The developer environment (JadeEnv) is the Jade IDE that we develop Jade on, connected to a server hosted on site that has both the backend running for the dev JAS and the web app of Simplifi.

When you make changes in the Jade IDE, you will be able to notice the changes on the JAS or the web app instantly, provided you refresh the screen.
