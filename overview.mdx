## Introduction

Welcome to the documentation for the Simplifi API!

Simplifi's public (REST) API provides your organisation with an alternative way to programmatically access and manage your organisation's data via HTTPS requests. This makes it simple to add automation functionality, such as pulling all your employees from your organisation, timesheets, pay periods and more.

## API Versioning

The Simplifi API is versioned from V1. Add /V1 to your Base URL to access the base Simplifi API.
