---
title: "How to Make a User Inactive"
---

<Frame>
  ![Image 46 1024x804](/images/image-46-1024x804.png)
</Frame>

1. Select **Our Profile**
2. Select **Our Users**
3. Select **Edit** next to the user you wish to inactivate
4. Switch the **‘Active User’** toggle to **NO**
5. Select **Save User Details**
