---

<Frame>
  ![Image 51 1024x516(1)](/images/image-51-1024x516\(1\).png)
</Frame>

1. Select **Our Profile**
2. Select **Our Users**
3. Select **Managers**
4. Use the dropdown list of your users to select the **Store Manager**
5. Click **Save Managers**
