---
title: "How to Reactivate a User"
---

<Frame>
  ![Image 47 1024x811](/images/image-47-1024x811.png)
</Frame>

1. Select **Our Profile**
2. Select **Our Users**
3. Switch the toggle ‘**Show inactive users too?**’ to **YES**. (The list of users below will now include inactive users.)
4. Select **Edit** next to the user you wish to activate
5. Switch the **‘Active User’** toggle to **YES**
6. Select **Save User Details**
