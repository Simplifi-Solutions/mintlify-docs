## About

The "System" refers to the actual Simplifi application itself. In this module you can modify settings for Simplifi itself.

## Setting Up Guide

You can follow this guide to set up the system. This guide will run through heading by heading and detail what the settings are, what they do and provide an example of the setting. If a setting is complicated, it will lead to another page.

<Warning>
  System Settings that are self-explanatory such as Password Control, Facebook etc are not included.
</Warning>

1. **Test Server Email Exceptions**

   Receiving emails from the test server does not work by default. If you would like to, for example, test the email function via the test server, you would want to add your email to this list. This would let the server still send you the email.
