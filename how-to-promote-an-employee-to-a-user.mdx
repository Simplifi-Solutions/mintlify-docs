title: "How to Promote an Employee to a User"
---

This will allow this employee to log in as both a staff member and as a company user.

<Frame>
  ![Image 39 1024x836](/images/image-39-1024x836.png)
</Frame>

1. Select **Staff**
2. Search for the employee’s name
3. Select the employee
4. Switch the ‘**Company User**‘ toggle to **YES**
5. Click **Save / Update Details**

<Frame>
  ![Image 40](/images/image-40.png)

  1. Click **Yes**
</Frame>

<Frame>
  ![Image 41 1024x802](/images/image-41-1024x802.png)
</Frame>

1. Select **Our Profile**
2. Select **Our Users**
3. Select **Edit** next to the user you wish to edit
4. Use the form here to grant or restrict access
5. Click **Save User Details**
