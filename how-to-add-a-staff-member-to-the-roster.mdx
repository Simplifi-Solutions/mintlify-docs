---
title: "How to Add a Staff Member to the Roster"
---

<Frame>
  ![Roster1 1 1024x401](/images/roster1-1-1024x401.png)
</Frame>

1. Select Rostering
2. Select the Daily View
3. Choose the Start timeslot next to the role you wish to add
4. Select the green plus next to the person you wish to add

<Frame>
  ![Image 4 1024x349](/images/image-4-1024x349.png)
</Frame>

The shift is added from the start time you selected but to the end of the day

1. Select the shift
2. Enter the end time here
3. Select Save

The shift is added with the default activity set for the role, with a lunch break if entitled. You may wish to add or change these timesheet activities. To do so:

<Frame>
  ![Image 3 1024x349(1)](/images/image-3-1024x349\(1\).png)
</Frame>

1. Select the shift bar
2. Select the clock icon to bring up the timesheet

<Frame>
  ![R2(1)](/images/r2\(1\).png)
</Frame>

1. Click this + button to add an activity to the top of the list
2. Select one of these + buttons to add an activity directly below
3. Choose from the list of activities
4. Enter the finish time of this activity
5. If you wish to notify the staff member of the change, switch this toggle to a yes
6. Select Save Details.
