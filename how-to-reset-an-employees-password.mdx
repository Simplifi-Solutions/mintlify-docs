---
title: "How to Reset an Employee's Password"
---

<Frame>
  ![Image 32 1024x455](/images/image-32-1024x455.png)
</Frame>

1. Click **Staff** (If you can not see this ‘Staff’ button, you will need to ask your store administrator to grant you access)
2. Type in the name or partial name of the employee
3. Click the **Search Icon**
4. Select the **Employee**
5. Select **Reset Password**

**A. Please note that you are only able to reset the password for employees who use a personal email address that is not associated with the company single sign-on**

<Frame>
  ![Image 33](/images/image-33.png)
</Frame>

1. Select – **Yes - Reset their Password**

<Frame>
  ![Image 34](/images/image-34.webp)
</Frame>

1. Select **OK**

The password will be reset to the employee’s first name in lowercase.
