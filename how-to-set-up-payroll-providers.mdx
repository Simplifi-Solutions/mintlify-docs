You cannot set up a new payroll provider, but you may change a payroll provider's settings.

**Main Settings**

| Setting Name                 | Description                                                                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| Available Within Industries: | Here you can select what industries support this payroll provider                                                                |
| Payroll Code Questions:      | Here you can select what questions must be answered for the payroll provider to function.  This is mostly just relevant to EdPay |
| Payroll Activities           | Payroll activities are activities other than just work that can be substituted in. One of the most common of these is sick leave   |

**Activity Settings**

| Setting Name          | Description                                                                                                                                                                                                                           |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Payroll Code          | The payroll code is a numerical or non-numerical code to let the payroll provider identify the relevant activity on their system. **You do not always have to enter one in. **Check whether the payroll provider requires this or not |
| Description           | This is the name of the activity.                                                                                                                                                                                                     |
| Pay Rate Factor       | Factor to multiply the pay rate by.                                                                                                                                                                                                   |
| Display Colour        | This is the colour that is displayed with this activity. Enter in a colour using **Hexadecimal**                                                                                                                                      |
| PayRoll Category Code | ????                                                                                                                                                                                                                                  |
| Archived              | Select this if the activity is no longer in use                                                                                                                                                                                       |

Once you are done editing a payroll provider make sure to press **Save Activity Details** to save your work!
