## About

The Job Queue is how Simplifi manages scheduled and requested jobs.

Jobs are anything that the user asks the server to do. Jobs either are requested instantaneously by a user, such as asking Simplifi to generate an excel file download, or are scheduled by a user or somebody from our team.
