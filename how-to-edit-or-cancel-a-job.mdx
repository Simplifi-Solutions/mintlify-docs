---
title: "How to Edit or Cancel a Job"
---

<Frame>
  ![2024 05 28 11 48 542 1024x495](/images/2024-05-28_11-48-542-1024x495.png)
</Frame>

1. Select the Calendar
2. Select the Date
3. Select the Job
4. Select Edit Job Details to Edit – Once you have edited the job make sure to go to the final step and click ‘Update this Job for me’\
   **or**
5. If you wish to cancel the job – Select ‘Cancel This Job’
