## What is JAS?

The JAS (Jade Administration Software) is the backend of Simplifi. The main purpose of the system is to set up Simplifi, organisations within Simplifi, and monitor Simplifi to ensure it is running smoothly for our clients.

As you can see below in the diagram, this is our **private**, admin software. It lets you interact with the Simplifi servers and modify settings for the organisations that use our software from behind the scenes.

## Navigating the UI

Navigate the main tabs using the UI at the top. This UI is laid out to give you easy access to the main 'modules' of Simplifi.

## Home Screen

<Frame>
  ![Image](/image-5.png)
</Frame>

The Home Screen of the JAS is what you will see when you first open up the application. You have a **Quick Search** at the top, where you can easily search for users or organisations.

**Where was I?** below the quick search shows the last user/organisation that you accessed.

**Actionable Items** on the right which shows jobs that need to be added to the [job queue](/job-queue), missing name matches (teacher names not matching what is on the NZ Teachers' Registry) and other [common errors](/common-errors).

The **Daily Summary** has statistics about Simplifi such as the number of staff, average request times etc. This can be useful to ascertain whether the server is performing normally. It also displays the last 100 requests to the server which you see in the table taking up most of the screen.
